Open source, with a clear path to production deployment.

Some teams want to self-host. Others want a managed path with clear documentation, controlled deployment, and a cleaner integration model.

1
Clone and initialize

Download the complete health system with all anatomical tracking files and processing tools.

# Clone the EIR Health System
git clone https://github.com/Eir-Space/health.git
cd health

# Generate complete health structure (176 files)
python3 eir-parser/anatomical_structure_generator.py
2
Configure agent integration

Add health consciousness to your agent's core personality system.

# Add to your agent's SOUL.md or equivalent
## Health Context Loading
Every session, read:
1. health/active.md - Current health support needs
2. health/diagnosis/ - Active conditions affecting daily function  
3. Relevant rehabilitation protocols for ongoing conditions
3
Deploy health-aware responses

Agent automatically becomes health-conscious with context-appropriate responses.

Example: Context-aware response

User: "I need to work on a long document"
Agent: "With your finger injury, let's plan this strategically:
       - 25-minute typing sessions with 5-minute exercise breaks
       - Voice dictation backup for extended sections
       - Ergonomic positioning to minimize strain
       Ready for your next rehab session?"
