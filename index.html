Installation

Open source, with a clean path from clone to operational use.

The installation flow is intentionally direct: clone the repo, generate the structure if needed, then wire the active files into your agent context.

1

Clone and initialize

Pull the repository and generate the full anatomical structure so the workspace contains the complete health tracking surface. 

git clone https://github.com/Eir-Space/health.git
cd health

python3 eir-parser/anatomical_structure_generator.py
2

Configure agent integration

Point your agent to the files that should shape its response behavior before each session or task run. 

## Health Context Loading
Every session, read:
1. health/active.md
2. health/diagnosis/
3. Relevant rehabilitation protocols
3

Deploy health-aware responses

Once loaded, the agent can adapt ordinary planning or work support to current injuries, fatigue, rehab cadence, and safety constraints. 

User: "I need to work on a long document"

Agent: "With your finger injury, let's plan this strategically:
- 25-minute typing sessions with 5-minute exercise breaks
- Voice dictation backup for extended sections
- Ergonomic positioning to minimize strain"
