# Eir Space Health Skill

`health` is a base healthcare skill for agents that use the Health.md standard.

It helps an agent:
- store and update user health information in `health.md`
- ask follow-up questions and persist information gaps
- separate confirmed facts from unconfirmed findings
- attach condition/context skills such as `pregnancy` and `diabetes`

## Skill Folder

- `health/` - the `health` skill (`SKILL.md`, `agents/openai.yaml`, references)

## Intended Use

This repository is intended for publication to skill directories (for example `skills.sh`) and for use in Eir-compatible agents.
